# 🚀 SkillPath AI

SkillPath AI is an AI-powered career roadmap generator that helps learners create personalized learning paths for their desired career goals. Using Google's Gemini AI, the platform generates structured learning roadmaps, recommends resources, suggests projects, and helps users track their learning progress.

## 🌟 Features

### 🤖 AI-Powered Roadmap Generation

* Generates personalized learning roadmaps using Gemini AI
* Tailored according to:

  * Career Goal
  * Skill Level
  * Study Hours per Week
  * Free Resources Preference

### 📚 Structured Learning Path

* Multiple learning phases
* Skills to learn in each phase
* Suggested projects for practical experience
* Curated learning resources

### 🎯 Career Guidance

* AI-generated roadmap overview
* Career outcome prediction
* Estimated learning duration

### 📈 Progress Tracking

* Mark completed phases
* Progress percentage calculation
* Data persisted using Local Storage

### 💾 Save Roadmaps

* Automatically save generated roadmaps
* Access previous roadmaps anytime

### 📄 PDF Export

* Download generated roadmaps as PDF documents

### 🌙 Dark Mode

* Toggle between Light and Dark themes

### 📱 Responsive Design

* Works across desktop and mobile devices

## 🎮 How It Works

1. Enter your desired career goal.
2. Select your current skill level.
3. Enter study hours available per week.
4. Choose whether you want only free resources.
5. Generate your AI-powered roadmap.
6. Track progress through roadmap phases.
7. Save and revisit roadmaps anytime.
8. Export roadmaps as PDF documents.
